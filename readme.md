# bouwekit

A prototype for text based UI built with React Ink.

## Development

Open a Terminal tab and run:

```
npm run dev
```

Open another Terminal, where after each code change, you run:

```
node dist/cli.js
```

## Install

```bash
$ npm install --global bouwekit
```

## CLI

```
$ bouwekit --help

  Usage
    $ bouwekit
```
